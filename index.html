Research

My research work

Leveraging web scraping and stacking ensemble machine learning techniques to enhance detection of major depressive disorder from social media posts

This study explores machine learning and deep learning techniques for detecting Major Depressive Disorder (MDD) through social media text analysis. Data was collected using web scraping and publicly available Twitter and Reddit datasets. Key findings show that the LSTM model achieved 93.72% accuracy on Reddit, 99.85% on Twitter, and 96.47% on a web-scraped dataset using a Stacking Ensemble approach. The model was evaluated using accuracy, precision, recall, and F1-score, demonstrating its potential in early MDD detection and intervention.

Core Techniques: Web Scraping, NLP, Sentiment Analysis, Deep Learning, Hyperparameter Optimization, Stacking Ensemble
